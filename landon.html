Landon Shopping Website

A full-featured shopping website built with Java and Spring Boot framework for a seamless online shopping experience.

Problem I Solved

Users needed a responsive and reliable shopping platform that allows them to browse products, make purchases, and manage their orders easily. Existing platforms for this niche were either slow, difficult to navigate, or lacked essential features like cart management and search functionality.

Features

  • Responsive product catalog with search and filters
  • Shopping cart and checkout system
  • User authentication and profile management
  • Order tracking and history
  • Admin panel for product management

Technologies Used

Java, Spring Boot, MySQL, Thymeleaf, HTML, CSS, JavaScript

Screenshots

What I Learned

Through this project, I gained experience with backend development using Spring Boot, integrating front-end templates with dynamic data, and deploying a full-stack application. I also improved my understanding of user experience design and responsive layouts.
